# Messaging Framework Generator

This skill creates a messaging framework to guide all marketing and sales communications.

## Instructions

### Step 1: Review Brand and Customer Insights

- Review the brand positioning, brand story, and customer avatar.
- Ensure a deep understanding of the brand's unique value proposition and the customer's needs and pain points.

### Step 2: Develop Key Messaging Pillars

- Identify 3-5 key messaging pillars that represent the core themes and ideas the brand wants to communicate.
- Each pillar should be a high-level concept that supports the brand positioning.

### Step 3: Create Supporting Messages

- For each messaging pillar, develop a set of supporting messages that provide more detail and evidence.
- These messages should be specific, benefit-oriented, and tailored to the target audience.

### Step 4: Craft a Tagline and Elevator Pitch

- **Tagline:** Create a short, memorable phrase that captures the essence of the brand.
- **Elevator Pitch:** Write a 30-60 second summary of the business, its value proposition, and its target audience.

### Step 5: Produce Deliverables

- Create a Markdown document titled `messaging_framework.md`.
- The document should include:
    - The key messaging pillars.
    - The supporting messages for each pillar.
    - The brand tagline.
    - The elevator pitch.
    - Guidelines on how to adapt the messaging for different channels and audiences.

## Examples

### Example Input

"I need to create a messaging framework for my new B2B SaaS product. It helps sales teams close more deals by providing real-time coaching."

### Example Output

A `messaging_framework.md` file that outlines a messaging framework for the SaaS product. The key messaging pillars might be: "Close More Deals," "Coach Your Reps to Success," and "Data-Driven Insights." The supporting messages would provide specific details about how the product helps sales teams achieve these outcomes. The tagline could be something like: "Real-time coaching for a winning sales team."
