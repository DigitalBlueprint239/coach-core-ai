# SEO Blog Article Writer

This skill helps businesses write blog articles that are optimized to rank on search engines.

## Instructions

### Step 1: Keyword Research

- Identify a primary keyword and 2-3 secondary keywords for the article.
- Use web search to analyze the search intent behind the keywords (are users looking for information, products, or services?).
- Analyze the top-ranking articles for the primary keyword to understand what type of content is performing well.

### Step 2: Create an SEO-Optimized Outline

- Structure the article with a clear hierarchy of headings (H1, H2, H3).
- Include the primary keyword in the H1 title and at least one H2 subheading.
- Sprinkle the primary and secondary keywords naturally throughout the outline.

### Step 3: Write the Blog Article

- Write a high-quality, informative, and engaging article that fully answers the user's search query.
- Naturally incorporate the primary and secondary keywords throughout the text.
- Aim for a word count that is similar to or longer than the top-ranking articles.
- Include a compelling introduction and a clear conclusion with a call-to-action.

### Step 4: Optimize for On-Page SEO

- **Title Tag:** Write a compelling title tag (under 60 characters) that includes the primary keyword.
- **Meta Description:** Write a concise meta description (under 160 characters) that summarizes the article and includes the primary keyword.
- **Image Alt Text:** Write descriptive alt text for all images, and include the primary keyword where appropriate.
- **Internal Linking:** Add 2-3 internal links to other relevant pages on the user's website.

### Step 5: Produce Deliverables

- Create a Markdown document titled `seo_blog_article.md`.
- The document should include:
    - The full text of the blog article.
    - The recommended title tag and meta description.
    - A list of recommended internal links.

## Examples

### Example Input

"I want to write a blog post about the benefits of yoga for beginners. My target keyword is 'yoga for beginners'."

### Example Output

A `seo_blog_article.md` file containing a well-written and SEO-optimized blog post about the benefits of yoga for beginners. The article would be structured with clear headings, include the target keyword and related keywords, and provide valuable information to the reader. The file would also include a recommended title tag, meta description, and internal links.
