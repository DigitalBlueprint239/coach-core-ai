# Hashtag Research Engine

This skill helps businesses find the best hashtags to use on their social media posts.

## Instructions

### Step 1: Identify Core Keywords

- Brainstorm a list of keywords that are relevant to the user's business, industry, and target audience.

### Step 2: Research Hashtag Variations

- Use social media platforms and third-party tools to find hashtag variations, including:
    - **Broad hashtags:** High-volume hashtags that are relevant to the industry (e.g., #marketing).
    - **Niche hashtags:** More specific hashtags that are relevant to the user's niche (e.g., #contentmarketingtips).
    - **Branded hashtags:** A unique hashtag for the user's business (e.g., #YourBrandName).
    - **Community hashtags:** Hashtags used by a specific community (e.g., #smallbusinessowner).

### Step 3: Analyze Hashtag Performance

- Analyze the performance of each hashtag, including the number of posts, engagement rate, and audience demographics.
- Choose a mix of hashtags with different levels of popularity.

### Step 4: Produce Deliverables

- Create a Markdown document titled `hashtag_recommendations.md`.
- The document should include:
    - A list of recommended hashtags, categorized by type (broad, niche, branded, community).
    - The estimated reach and engagement for each hashtag.
    - A recommended hashtag strategy for different types of content.

## Examples

### Example Input

"I need help finding the right hashtags for my fitness coaching business on Instagram."

### Example Output

A `hashtag_recommendations.md` file that provides a list of recommended hashtags for the fitness coaching business. The list would include a mix of broad hashtags (e.g., #fitness, #workout), niche hashtags (e.g., #personaltrainer, #onlinecoaching), and community hashtags (e.g., #fitfam, #gymmotivation). The file would also include a recommended strategy for using the hashtags on different types of posts.
