# YouTube Video Script Writer

This skill helps businesses write compelling scripts for their YouTube videos.

## Instructions

### Step 1: Define the Video Goal and Target Audience

- What is the goal of the video (e.g., educate, entertain, convert)?
- Who is the target audience for the video?
- What action do you want the viewer to take after watching the video?

### Step 2: Research and Outline the Video

- Research the topic to gather information, data, and examples.
- Create a detailed outline of the video, including:
    - **Hook:** A strong opening to grab the viewer's attention.
    - **Introduction:** Introduce the topic and what the viewer will learn.
    - **Main Content:** Break down the main points into a logical sequence.
    - **Conclusion:** Summarize the key takeaways and include a call-to-action.

### Step 3: Write the Video Script

- Write the script in a conversational and engaging tone.
- Read the script out loud to check for flow and timing.
- Add visual cues and suggestions for b-roll, graphics, and on-screen text.
- Aim for a script length that is appropriate for the video format (e.g., 5-10 minutes for a tutorial, 2-3 minutes for a product review).

### Step 4: Produce Deliverables

- Create a Markdown document titled `youtube_script.md`.
- The document should include:
    - The full video script, with clear sections for the hook, intro, main content, and conclusion.
    - Suggestions for visuals, music, and editing.
    - A recommended video title, description, and tags.

## Examples

### Example Input

"I want to create a YouTube video that explains how to use our new project management software."

### Example Output

A `youtube_script.md` file containing a detailed script for a tutorial video. The script would walk the viewer through the software's key features, with clear instructions and visual cues. The file would also include a recommended title, description, and tags that are optimized for YouTube search.
